# Security Policy

## Supported Versions

Security fixes are applied to the latest published version of `gs-app-finance-jg`.

## Reporting a Vulnerability

Please do **not** open public GitHub issues for security vulnerabilities.

Report vulnerabilities privately to the maintainers with:

- Clear description of the issue
- Reproduction steps
- Potential impact
- Any relevant logs/screenshots

If the report is valid, we will coordinate a fix and disclose responsibly.
